# FinSight

AI-powered financial assistant for modern businesses. FinSight unifies cash flow intelligence, automated insights, and reporting into a clean, fast dashboard that helps teams make confident decisions.

## Why FinSight

FinSight focuses on actionable intelligence rather than raw spreadsheets. It brings AI-driven insights, a natural-language chatbot, and ready-to-share reports together in one workflow so you can move faster on what matters.

## Most Valuable Features

- **AI Insights**
  - Detects anomalies and surfaces unusual spending patterns automatically.
  - Highlights key trends in revenue, expenses, and profitability.

- **AI Chatbot**
  - Ask plain-language questions about your finances and get instant answers.
  - Explore drivers behind changes (e.g., “Why is cash flow down this month?”).

- **Tax Summary**
  - Consolidated view of tax-relevant totals and breakdowns.
  - Helps track liabilities and stay audit-ready with clear summaries.

- **PDF Export**
  - One-click exports for reports you can share with investors or auditors.
  - Generates polished PDFs for monthly, yearly, or all-time ranges.

## What You Get

- Real-time transaction tracking and categorization
- KPI cards for revenue, expenses, and net profit
- Cash-flow trends with month/year labeling and amounts on the axis
- Tax summaries with clean, at-a-glance reporting
- AI insights plus a conversational assistant
- Professional PDF reports on demand

## Getting Started

### Prerequisites

- Node.js (recommended LTS)
- npm or yarn

### Install

1. Install dependencies for the client and server:

```bash
cd client
npm install

cd ../server
npm install
```

2. Start the development servers:

```bash
# in one terminal
cd server
npm run dev

# in another terminal
cd client
npm run dev
```

3. Open the client URL printed in your terminal.

## Usage Notes

- Use the **AI Insights** panel to surface unusual trends and anomalies.
- Ask questions using the **AI Chatbot** to understand your numbers quickly.
- Review liabilities and totals in **Tax Summary** for compliance visibility.
- Export any report as a **PDF** for sharing and record-keeping.
